# 🧮 Web Calculator

A simple and responsive calculator built using **HTML, CSS, and JavaScript**.  
This project performs basic arithmetic operations in the browser with a clean UI.

## 📌 About the Project

This is a beginner-friendly web project that demonstrates how to build a functional calculator using core web technologies:

- HTML → structure  
- CSS → styling  
- JavaScript → logic & calculations
